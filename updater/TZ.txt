Необходимые методы работы с YouTube API находятся в папке youtube_api

Для каждого канала (коллекция channels) необходимо сохранить: 
- Ссылки на соц. сети 
- Список ссылок плейлисты канала 
- Список понравившихся видео 
- Список favorites видео 
- Название канала
- Описание канала 
- Список подписок канала (на кого он подписан)
- URL иконки канала 

Для каждого видео (коллекция videos): 
- Список пользователей, оставивших комментарий          : get_comment_author_url_list
- Кол-во лайков/дислайков под видео 
- Ссылка на плейлист, в котором находится видео         : 
- Название видео
- Описание видео 
- Ссылка на миниатюру видео
- Время публикации 
- Категория
- Теги
- Длина 
- Кол-во просмотров
- Кол-во комментариев
- Список комментариев (для каждого: текст, ссылка на автора, рейтинг)
